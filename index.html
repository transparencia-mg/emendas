Informações sobre os Dados

Fonte dos Dados: Os dados são carregados automaticamente do repositório GitHub transparencia-mg/emendas
Atualização Automática: Os dados são atualizados automaticamente quando novos arquivos são adicionados ao repositório GitHub.
Arquivos Disponíveis:
  • dados_gerais_emendas.csv - Informações gerais sobre todas as emendas parlamentares
  • relatorio_de_indicacoes_transparencia.csv - Indicações de emendas por parlamentares
  • plano_execucao_emendas_pix.csv - Plano de execução das emendas na modalidade PIX
  • execucao_transferencias_especiais_pix.csv - Execução das transferências especiais via PIX
